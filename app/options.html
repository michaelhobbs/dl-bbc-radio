Options

By default dl-bbc-radio saves programs to the directory where the extension is installed. Usually this is the first option to configure.

dl-bbc-radio uses a get_iplayer preset (dl-bbc-radio) to store the configuration it should use. Here you can view the settings and edit them. See: get_iplayer documentation for available options.

For adding other get_iplayer settings to the recording command, use the get_iplayer CLI to add prefs for preset dl-bbc-radio or a text editor to modify the preset manually. eg:

get_iplayer --preset=dl-bbc-radio --prefs-add --no-artwork

get_iplayer options are documented on their wiki.
